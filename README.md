# UntilTheEnd — 人生の予算アプリ

> 最後の日まで、お金は足りる？ — 人生の収支シミュレーター

月の収入・出費・資産・利回り・年齢を入力すると、**何歳でお金がゼロになるか**を計算します。
資産が増える場合は **100歳までの収支を月別／年別の表**で確認でき、**逆算プラン**では
「○歳で○円残したい」から毎月使える金額を求められます。
**インフレ率・年金は任意入力**で、未入力なら自動的に計算から除外されます。

## 機能

- **資産シミュレーション** … 現在の年齢・月収・月支出・資産・年利回りから、資産ゼロになる年齢を算出
- **資産推移グラフ** … 〜100歳までの資産残高を SVG ラインチャートで可視化（ゼロ到達点を赤でマーク）
- **収支テーブル** … 年次／月次を切り替えて確認（最大100歳まで）
- **逆算プラン** … 死亡年齢と残したい資産から、毎月使える金額を逆算
- **任意オプション** … インフレ率（支出が毎年上昇）／年金（受給開始年齢から月額を加算）。空欄なら考慮しない

## 計算モデル

- 月複利：`毎月 資産 ×(1+月利) + (収入 − 支出)`
- 年利 → 月利：`(1 + 年利)^(1/12) − 1`
- インフレ：`支出ₘ = 基準支出 ×(1+インフレ率)^(m/12)`
- 年金：受給開始年齢以降、月の収入に年金額を加算
- 逆算は将来価値の総和から基準支出について線形に解く（インフレ・年金も厳密に反映）
- 逆算結果はインフレ入力時「今の物価基準」の月額

> ※ 税金は未考慮。インフレ率・年金は任意。あくまで目安です。

## 使い方

依存ゼロの静的アプリです。

```bash
node server.js   # http://localhost:3210 を開く
```

または `index.html` をブラウザで直接開くだけでも動作します。

## 構成

| ファイル | 役割 |
|----------|------|
| `index.html` | 画面とフォーム |
| `styles.css` | 配色（白・青・緑のヘルスケア系）とレイアウト |
| `app.js` | シミュレーション／逆算ロジックと描画 |
| `server.js` | 依存ゼロの静的ファイルサーバー |
